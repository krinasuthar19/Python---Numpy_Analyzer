# NumPy Analyzer

## Overview

NumPy Analyzer is an interactive Python-based tool designed to perform a wide range of array operations using the NumPy library. It provides a menu-driven interface that allows users to create arrays and perform mathematical, statistical, and analytical operations efficiently.

This project helps in understanding core NumPy concepts through practical implementation, including array manipulation, filtering, aggregation, and advanced computations.

---

## Features

- Create **1D, 2D, and 3D arrays**
- Perform **indexing and slicing**
- Apply **mathematical operations** (addition, subtraction, multiplication, division)
- Execute **matrix operations** (dot product, matrix multiplication)
- Combine and split arrays
- Search, sort, and filter array elements
- Compute **statistical measures** (mean, median, standard deviation, variance)
- Calculate **percentiles and correlation coefficients**
- Menu-driven interactive CLI interface

---

## Tech Stack

- Python
- NumPy

Install the required library:

## How to Run

1. Save the script as `numpy_analyzer.py`
2. Run the program:

```bash
python numpy_analyzer.py
```

3. Follow the menu instructions to perform operations
